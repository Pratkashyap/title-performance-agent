I built a 9-agent AI analytics team for Max APAC streaming.
Each agent has a specific role, a defined expertise, and one job to do.

My Team:

🧠 Orchestrator — reads your question, classifies it, routes to the right specialist
📊 Data Agent — writes the SQL, hits the database, no analyst needed
🔍 Performance Analyst — tells you exactly why a title is underperforming
📐 Benchmark Agent — compares performance against peer titles in the same genre
📈 Trend Agent — week-over-week momentum, which markets are growing, which are declining
🎭 Genre & Catalog Agent — platform-wide genre health and engagement gaps
👤 Subscriber Agent — links viewing behaviour to churn risk before it shows up in numbers
🚨 Alert Agent — proactive flags. finds problems before anyone has to ask
⚖️ Quality Critic — scores every answer before you see it. Below 8 out of 10 — it rewrites it.

Cost per query: ~$0.04
Runs on a laptop. No cloud. No SaaS.
Built with Python + Claude API.
